15. Algorithms III: Interior-Point Methods

Date: 2026-02-10 Duration: 90 min Tags: algorithms, interior-point

Overview: Barrier methods, the central path, self‑concordance, and primal‑dual methods.

Prerequisites: Lecture 14

Learning Objectives

After this lecture, you will be able to:

  • How to solve inequality-constrained problems using the barrier method.
  • The concept and properties of the central path.
  • The role of Newton's method in centering steps and the complexity implications of self-concordance.
  • The structure of Primal-Dual Interior-Point methods.

Key Concepts

1. Inequality Constrained Minimization

We solve $\min f_0(x)$ subject to $f_i(x) \le 0, i=1,\dots,m$ and $Ax=b$. The interior-point approach approximates this by an equality-constrained problem using a barrier function to enforce inequalities.

2. Logarithmic Barrier

The log-barrier is $\phi(x) = -\sum_{i=1}^m \log(-f_i(x))$. The approximated problem is $\min f_0(x) + (1/t)\phi(x)$ subject to $Ax=b$. As $t \to \infty$, the solution $x^*(t)$ converges to the optimal solution. The parameter $t$ controls the accuracy of the approximation.

3. The Central Path

The set of solutions $\{ x^*(t) : t > 0 \}$ forms the central path. Points on the central path satisfy the KKT conditions for the original problem with $\lambda_i^*(t) = -1/(t f_i(x^*(t)))$. The duality gap at $x^*(t)$ is exactly $m/t$.

4. Barrier Method

The algorithm iterates:

  1. Centering Step: Compute $x^*(t)$ by minimizing $t f_0 + \phi$ starting from current $x$, using Newton's method.
  2. Update: Increase $t$ by factor $\mu > 1$.
This continues until $m/t < \epsilon$. The number of centering steps is relatively small (e.g., 20-50), and each Newton step is efficient.

5. Complexity Analysis

If the barrier is self-concordant (like the log-barrier for linear/quadratic/SDP constraints), Newton's method converges quadratically. The total number of Newton iterations grows as $O(\sqrt{m} \log(1/\epsilon))$, proving polynomial-time complexity for LP, QP, GP, SDP.

6. Generalized Inequalities

For conic problems (SOCP, SDP), we use generalized log-barriers. For $X \succeq 0$, the barrier is $-\log\det(X)$. For $x \in \mathcal{K}_{soc}$, it is $-\log(t^2 - \|y\|^2)$. The theory of self-concordance extends seamlessly.

7. Primal-Dual Interior-Point Methods

Unlike the barrier method which keeps primal points strictly feasible and updates duals implicitly, Primal-Dual methods update primal ($x$) and dual ($\lambda, \nu$) variables simultaneously by linearizing the modified KKT conditions (perturbed complementary slackness). They are generally more efficient and robust, handling infeasible starts.

8. Feasibility Phase

Finding a strictly feasible starting point $x_0$ is necessary for the barrier method. This is done by solving a "Phase I" problem: minimize $s$ subject to $f_i(x) \le s$, $Ax=b$. If $s^* < 0$, we found a feasible point.

Example Problems

Example 15.1: Log-Barrier Gradient and Hessian

Problem: Derive $\nabla \phi(x)$ and $\nabla^2 \phi(x)$ for $\phi(x) = -\sum \log(b_i - a_i^\top x)$.

Solution:

Let $s_i = b_i - a_i^\top x$. Then $\phi = -\sum \log s_i$. Chain rule: $\nabla \phi = \sum (1/s_i) a_i = A^\top d$ where $d_i = 1/s_i$. Hessian: $\nabla^2 \phi = \sum (1/s_i^2) a_i a_i^\top = A^\top \text{diag}(d)^2 A$. This structure $A^\top D A$ is typical for interior-point methods.

Example 15.2: Central Path for 1D Problem

Problem: $\min x$ subject to $x \ge 0$. Find central path.

Solution:

Barrier problem: $\min t x - \log x$. Optimality: $t - 1/x = 0 \implies x^*(t) = 1/t$. As $t \to \infty$, $x^*(t) \to 0$, which is the optimal solution. Duality gap $1 \times (1/t) \times 1 = 1/t$. Matches theory.

Example 15.3: Analytic Center of Cube

Problem: Find analytic center of unit cube $-1 \le x_i \le 1$.

Solution:

Constraints: $1-x_i \ge 0, 1+x_i \ge 0$. Barrier $\phi(x) = -\sum (\log(1-x_i) + \log(1+x_i)) = -\sum \log(1-x_i^2)$. Gradient $\nabla \phi_i = \frac{2x_i}{1-x_i^2}$. Set to 0 implies $x_i=0$. Center is the origin.

Example 15.4: Duality Gap Property

Problem: Prove gap is $m/t$ for $x^*(t)$.

Solution:

Lagrangian of original: $L(x, \lambda) = f_0(x) + \sum \lambda_i f_i(x)$. Condition for $x^*(t)$: $\nabla f_0 + \sum \frac{-1}{t f_i} \nabla f_i = 0$. Let $\lambda_i^*(t) = -1/(t f_i(x^*(t)))$. This $\lambda^*$ is dual feasible since $f_i < 0 \implies \lambda_i > 0$. Gap: $f_0(x) - g(\lambda) = f_0(x) - (f_0(x) + \sum \lambda_i f_i(x)) = -\sum \lambda_i f_i(x)$. Substitute $\lambda_i^*$: $-\sum (-1/(t f_i)) f_i = \sum (1/t) = m/t$.

Example 15.5: Phase I Method

Problem: Formulate Phase I for $Ax \le b$.

Solution:

Minimize $s$ subject to $a_i^\top x - b_i \le s$. This is always strictly feasible for any $x_0$ by choosing $s_0 > \max (a_i^\top x_0 - b_i)$. Use barrier method to solve Phase I. If optimal $s^* < 0$, we have a strictly feasible point for Phase II.

Example 15.6: Newton Step Cost

Problem: Cost of computing Newton step for LP barrier with $A \in \mathbb{R}^{m \times n}$.

Solution:

Hessian is $H = A^\top D A + \text{regularizer}$. Forming $H$: $O(mn^2)$ if dense. Solving $H \Delta x = -g$: $O(n^3)$. Total dominated by $O(mn^2)$. Much faster if $A$ is sparse.

Example 15.7: Primal-Dual System

Problem: Write the perturbed KKT conditions for Primal-Dual method.

Solution:

Standard form LP: $\min c^\top x$ s.t. $Ax=b, x \ge 0$. Conditions: 1. $A^\top y + s - c = 0$ (Dual feasibility) 2. $Ax - b = 0$ (Primal feasibility) 3. $x_i s_i = 1/t$ (Perturbed complementarity) We linearize this nonlinear system to find $(\Delta x, \Delta y, \Delta s)$.

Example 15.8: SDP Barrier Derivative

Problem: Derivative of $-\log\det(X)$.

Solution:

Differential $d(-\log\det X) = -\text{tr}(X^{-1} dX)$. So gradient $\nabla \phi(X) = -X^{-1}$. Second differential: $d(-X^{-1}) = X^{-1} (dX) X^{-1}$. Hessian action on direction $H$: $X^{-1} H X^{-1}$. This matches $1/x^2$ for scalar case.

Exercises

  1. Central Path Convergence:
    Prove that if the feasible set is bounded, the central path $\{x^*(t)\}$ has a limit point as $t \to \infty$ which is optimal.
    Solution: Since $f_0(x^*(t)) - p^* \le m/t$, as $t \to \infty$, value converges to $p^*$. Since set is compact, sequence has accumulation points. Continuity of $f_0$ implies any accumulation point is optimal. Uniqueness of central path (due to strict convexity of barrier) implies convergence to a specific optimal point (usually analytic center of optimal face).
  2. Gradient of Log-Barrier:
    Verify $\nabla \phi(x) = \sum_{i=1}^m \frac{1}{-f_i(x)} \nabla f_i(x)$.
    Solution: $\phi(x) = -\sum \log(-f_i(x))$. $\frac{\partial \phi}{\partial x} = -\sum \frac{1}{-f_i(x)} \frac{\partial (-f_i)}{\partial x} = -\sum \frac{1}{-f_i(x)} (-\nabla f_i(x)) = \sum \frac{1}{-f_i(x)} \nabla f_i(x)$.
  3. Complexity Bound:
    Justify the iteration bound $O(\sqrt{m} \log(1/\epsilon))$.
    Solution: Self-concordance theory guarantees that if $t_{k+1} = \mu t_k$, we stay in the "quadratic convergence basin" of Newton's method for the new $t$ if $\mu$ is small enough (specifically $1 + 1/\sqrt{m}$). Number of steps to increase $t$ from $t_0$ to $m/\epsilon$ is $\log_\mu (m/\epsilon t_0) \approx \sqrt{m} \log(m/\epsilon)$. Each centering takes $O(1)$ Newton steps.
  4. LP Central Path:
    For $\min c^\top x$ s.t. $x \ge 0$, find the central path.
    Solution: Problem is unbounded unless $c \ge 0$. Barrier $\min t c^\top x - \sum \log x_i$. $t c_i - 1/x_i = 0 \implies x_i = 1/(t c_i)$. Path is hyperbola $x_i(t) = 1/(t c_i)$. Converges to 0 (optimum) as $t \to \infty$ assuming $c_i > 0$.
  5. Primal-Dual Direction:
    Show that the primal-dual direction reduces to the pure Newton centering direction if $x$ is feasible and $s = -1/(t f_i)$.
    Solution: PD system linearizes $x_i s_i - \tau = 0$. If we are on central path, residuals are 0. Then $\Delta x, \Delta s$ are corrections for new target $\tau' < \tau$. The equations become essentially the same as linearizing optimality conditions for next $t$, which is what barrier method does.
